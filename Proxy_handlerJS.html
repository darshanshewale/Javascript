Javascript proxy

        Javascript proxy is an object which intercepts another object and resists the fundamental operations on it.
        This object is mostly used when we want to hide information abut one object from unauthorized access.

        A  proxy consists of two parts which are its target and handler. A target is a javascript object on which the 
        proxy is applied and the handler object contains the function to intercept any other operaration on it
