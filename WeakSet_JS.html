Javascript Weakset

        javascript WeakSet is used to store a collection of objects. It adapts the same properties of that of a set i.e does not
        store duplicates.The major difference of a weakset with a set is that a weakset is a collection of  objects and not values of 
        some particular type

        Parameterparamter oobject is an iterable object All the elements of the iterable object are added to the weakset.

        return type: it returns a weakset object
