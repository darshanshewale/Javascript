Let

        Javascript let is keyword used to declare variables that are block scoped variables defined with the let keyword can not be redeclared 
        and must be declared before use.

        let keyword was added in the es6 or es2015 version of javascript generally it is suggest that we  use the let keyword while working with 
        javascript

Block scoped 

    The variable which are declared inside the {} block are known as block-scoped variables  declared by the var keyword can not be 
    block-scoped

Global Scope

    A global scope variable is a variable declared in the main body of the source code, outside all functions

Function Scope

    A function scope variable is a variable declared inside  a function and can not be accessed outside the function

redeclaring variables in different blocks

The variables declared using let can be redeclared inside other blocks

redeclaring variables in the same block

We can not redeclare variable using the let keyword inside the same block it will throw an error

Does not support Hoisting

    The behaviour of moving the declarations on top of the script is know as hoisting
