Debugging in Javascript

        Using debugger keyword: The debugger keyword is used in the code to force stop the execution of the code at a breaking point and calls
        the debugging function. The debugger function is executed if any debugging is needed at all else no action is performed 

        Use of console.log() method there is another way in which the JS value can be displayed in the debugger window 

        setting break point: The console.log( is good way to debug errors but setting breakpoint is a faster more efficient and better method
        In this method Breakpoint are set in code which stops the excetution of code at that point so that the values of variables can be examined
        at that time. Here are some advantages of using Breakpoint over the console.log() method
