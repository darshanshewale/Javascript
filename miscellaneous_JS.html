Javascript Boolean

        Boolean is a datatype that returns either of two values i.e true or false In javascript,Boolean is used as a function to get the value 
        of variable,object,condition,expressions etc in terms of true or false

        Note:A variable or object which has a value is treated as a true boolean value,'0','NaN', empty string,'undefined' and 'NULL' is treated
        as false boolean variable
